# Restore Marketing — Caption + CTA Templates

Captions do 3 jobs, in order: **repeat the hook**, **deliver the value
in text form** (for the muted scroller who still needs the point),
**capture the DM**. Everything else is a hashtag block.

Reference: `RESTORE_STYLE_PROFILE.md` — voice rules + hashtag banks.

---

## Master caption skeleton (fits every template)

```
{hook line, ≤8 words}

{one-line promise / who this is for}

{the value in 2–5 tight lines — one number, one move, one truth}

Comment {KEYWORD} — I'll DM you the {asset}.

—
{hashtag block, 8–15 tags, rotated}
```

Rules:
- Line breaks are load-bearing. Instagram cuts the caption at ~125 chars
  in feed. Front-load the hook.
- No emoji stuffing. Max 2 per caption.
- CTA is one line, one keyword, all-caps.
- Sign-off `—` optional.

## Caption variations to test
Every strong hook deserves 3 caption tests. Rotate:

**Variation A — long form (educational).** 4–5 lines of value in caption
before the CTA. Best for SAVES and shares.

**Variation B — punch (mystery).** Only the hook + CTA. The value lives
in the video. Best for RETENTION + comment volume.

**Variation C — story wrapper.** 1 personal line ("last week we…"),
then hook, then CTA. Best for TRUST + follows.

Save the A/B/C results in `content/reference/restore/caption-tests.md`.

## First-comment (pinned) template
Always pin a first comment on every reel. Format:

```
DMs open — comment {KEYWORD} above and I'll send it within the hour. 
Also — save this for later. 📌
```

Rules:
- Include the keyword verbatim (helps DM automation catch the flow).
- "Save this for later" boosts save-rate.
- Reply to every comment in the first 60 min — response speed
  correlates with reach 1:1 for the next 6 hours.

## Hashtag rotation policy
- 8–15 hashtags per post (research: 10 is the sweet spot).
- Never post the same block twice in a row.
- Mix: **3 vertical (agency/local/reputation/ecom) + 3 topic (SEO/GBP/
  ads/email) + 2 wide (marketing/entrepreneur) + 1 branded (#restoremarketing).**
- Never use banned or shadow-banned tags. Audit quarterly via the
  Instagram search tool.
- Rotate 4 blocks per week — one per template family.
