# Restore Marketing — Hook Library

200+ hooks sorted by angle. Grab one, plug into the matching template in
`RESTORE_VIDEO_TEMPLATES.md`. **Hooks stay < 8 words, one specific noun,
one specific number wherever possible.**

Every hook obeys these rules:
- Voice hook lands by 1.2s.
- On-screen text hook lands by 1.5s and does NOT match the voice word-for-word.
- Specific number > adjective. "Booked 34 calls" > "insane results."
- No "stop scrolling," "hey guys," "nobody's talking about this" as opener (fatigued in 2026).

Format: `H### | HOOK LINE` — paired ON-SCREEN TEXT (all caps, ≤5 words) below.

## How to use this file
1. Every time you shoot a batch, print the section that matches your
   week's angle. Read 10 hooks aloud. Pick the 5 that punch hardest.
2. Rotate sections weekly so the account doesn't feel one-note.
3. When a hook lands (>60% avg watch, high saves), mark it here with a
   ⭐ so we double down on that angle next month.
4. Replace or archive any hook with <45% avg-watch after 72h.
